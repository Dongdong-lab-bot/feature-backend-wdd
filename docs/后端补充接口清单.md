## 2. 模块--缺失接口--接口作用

## P0（优先补）

- 登录与会话（reg_app / exec_app） -- POST /auth/sms/send -- 发送短信验证码（登录/注册/找回密码场景）
- 登录与会话（reg_app / exec_app） -- POST /auth/login/sms -- 手机验证码登录
- 登录与会话（reg_app / exec_app） -- POST /auth/register -- 注册新用户并建立租户/角色上下文
- 登录与会话（reg_app / exec_app） -- POST /auth/password/verify-code -- 校验找回密码验证码并签发 resetToken
- 登录与会话（reg_app / exec_app） -- POST /auth/password/reset -- 使用 resetToken 重置密码
- 登录与会话（reg_app / exec_app） -- GET /auth/me -- 登录后获取用户基础信息（非权限）
- 登录与会话（reg_app / exec_app） -- POST /auth/logout -- 服务端回收 refreshToken / 会话
- 登录兼容（监管端 Web） -- POST /api/auth/login（兼容）或统一改前端为 /auth/login -- 解决当前登录路径不一致
- 视频集控中心 -- GET /video/cameras/tree -- 获取食堂/区域下摄像头树（替代前端模拟摄像头）
- 视频集控中心 -- POST /video/streams/live-url -- 获取实时流播放地址
- 视频集控中心 -- POST /video/streams/playback-url -- 获取录像回看地址（按时间段）
- 视频集控中心 -- POST /video/snapshots -- 抓拍并落库
- 视频集控中心 -- POST /video/inspections -- 提交视频巡检结果（分值、问题、图片）
- 视频集控中心 -- GET /video/inspections -- 查询视频巡检记录列表
- 视频集控中心 -- GET /video/inspections/{id} -- 查询视频巡检详情
- 视频集控中心 -- GET /video/scores/statistics -- 分数统计页查询

## P1（业务闭环）

- 巡检模板能力（日/周/联/视频共用） -- PATCH /ledger/templates/{id}/active -- 模板启停（是否启用）
- 巡检模板能力（日/周/联/视频共用） -- PUT /ledger/templates/{id}/scope/users -- 保存模板覆盖人员
- 巡检模板能力（日/周/联/视频共用） -- PUT /ledger/templates/{id}/scope/canteens -- 保存模板覆盖食堂
- 巡检模板能力（日/周/联/视频共用） -- PUT /ledger/templates/{id}/score-schema -- 保存打分项/问题类型/整改要求
- 巡检模板能力（日/周/联/视频共用） -- GET /ledger/templates/{id} -- 编辑页回填模板完整配置
- 巡检记录能力（日/周/联/视频共用） -- GET /ledger/instances/{id} -- 记录详情页读取完整内容
- 巡检记录能力（日/周/联/视频共用） -- POST /ledger/instances/{id}/cancel -- 取消记录/撤销流程
- 巡检记录能力（日/周/联/视频共用） -- GET /ledger/instances/{id}/export -- 导出单条记录（PDF/Excel）
- 电子台账中心（监管端 Web） -- PUT /ledger/templates/{id}（补充字段：description/scope/extraConfig） -- 支持模板编辑页真实保存
- 电子台账中心（监管端 Web） -- GET /ledger/templates/{id}/scope -- 模板覆盖范围回显（人员、食堂）
- 电子台账中心（监管端 Web） -- POST /ledger/templates/{id}/scope/users:batch -- 批量覆盖人员
- 电子台账中心（监管端 Web） -- POST /ledger/templates/{id}/scope/canteens:batch -- 批量覆盖食堂

## P2（执行端能力补齐）

- 执行端设备中心 -- GET /device/list -- 设备列表查询
- 执行端设备中心 -- POST /device/bind -- 绑定设备
- 执行端设备中心 -- DELETE /device/unbind/{id} -- 解绑设备
- 执行端设备中心 -- PUT /device -- 更新设备信息
- 执行端员工管理 -- GET /employee/list -- 员工列表查询
- 执行端员工管理 -- GET /employee/{id} -- 员工详情
- 执行端员工管理 -- POST /employee -- 新增员工
- 执行端员工管理 -- PUT /employee -- 编辑员工
- 执行端员工管理 -- DELETE /employee/{id} -- 删除员工
- 执行端员工管理 -- PUT /employee/{id}/status -- 员工启停
- 执行端任务中心 -- GET /ledger/tasks/pending -- 日常待提交台账任务列表
