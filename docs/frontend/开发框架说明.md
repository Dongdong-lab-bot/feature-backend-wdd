## 三、核心功能详解

### 3.1 路由系统 (`router/index.ts`)

**功能**：

- 路由注册（后续开发者在此添加新路由）
- 全局前置守卫（登录检查、标题设置）
- 动态路由加载（懒加载优化）

**使用示例**：

```typescript
// 后续开发者添加新路由
const routes: RouteRecordRaw[] = [
  {
    path: '/dashboard',
    name: 'Dashboard',
    component: () => import('@/views/dashboard/index.vue'),
    meta: { title: '数据驾驶舱', requireAuth: true }
  }
]
```

### 3.2 用户状态管理 (`store/user.ts`)

**功能**：

- Token 存储与持久化（localStorage + Pinia）
- 用户信息管理（id、username、nickname、permissions、tenantId）
- 登录/登出方法
- 权限检查方法（`hasPermission`、`hasAnyPermission`、`hasAllPermissions`）

**使用示例**：

```typescript
import { useUserStore } from '@/store/user'

const userStore = useUserStore()

// 登录
userStore.login('token_value', {
  id: '123',
  username: 'admin',
  nickname: '管理员',
  permissions: ['user:add', 'user:edit'],
  tenantId: 'tenant_001'
})

// 权限检查
if (userStore.hasPermission('user:add')) {
  // 显示新增按钮
}

// 登出
userStore.logout()
```

### 3.3 HTTP 请求封装 (`utils/request.ts`)

**功能**：

- 自动注入 Token（`Authorization: Bearer ${token}`）
- 自动注入租户ID（`X-Tenant-ID`）
- 401 自动处理（清除登录信息 + 提示）
- 统一错误提示（ElMessage）
- 便捷方法导出（`get`、`post`、`put`、`del`）

**使用示例**：

```typescript
import { get, post } from '@/utils/request'

// GET 请求
const users = await get<User[]>('/api/users')

// POST 请求
const result = await post('/api/login', { username, password })
```

## 五、开发指南

### 5.1 启动项目

```bash
# 监管端
cd frontend/web-admin
npm install
npm run dev  # 访问 http://localhost:3000

# 执行端
cd frontend/web-execution
npm install
npm run dev  # 访问 http://localhost:3001
```

### 5.2 添加新页面

1. 在 `src/views/` 下创建页面组件
2. 在 `router/index.ts` 中注册路由
3. 在 `meta` 中配置标题和权限要求

**示例**：

```typescript
// src/views/user/list.vue
<template>
  <div>用户列表页面</div>
</template>

// router/index.ts
{
  path: '/user/list',
  name: 'UserList',
  component: () => import('@/views/user/list.vue'),
  meta: { 
    title: '用户管理', 
    requireAuth: true 
  }
}
```

### 5.3 调用后端接口

```typescript
import { get, post } from '@/utils/request'

// 获取用户列表
const getUserList = async () => {
  const data = await get<User[]>('/api/users')
  return data
}

// 创建用户
const createUser = async (user: User) => {
  const result = await post('/api/users', user)
  return result
}
```

## 六、注意事项

1. **Token 管理**：由 `store/user.ts` 统一管理，无需手动操作 localStorage
2. **路由守卫**：已配置全局前置守卫，新增路由无需重复编写登录检查
3. **请求拦截**：401 自动登出，其他错误自动提示，无需手动处理
4. **路径别名**：使用 `@/` 代替 `src/`，例如 `import { xxx } from '@/utils/request'`
5. **权限标识符**：参考 `permissions.md`，严格按照命名规范（模块名:操作名）
