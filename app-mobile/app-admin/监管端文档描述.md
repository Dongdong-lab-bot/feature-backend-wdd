# App Admin 项目文件架构文档

本文档详细说明了 `app-mobile/app-admin` 项目的目录结构及各页面的功能用途。

## 目录结构概览

```
src/pages/
├── login/                  # 登录与认证模块
├── index/                  # 首页与全局排名模块
├── daily/                  # 日管控模块
├── weekly/                 # 周排查模块
├── monthly/                # 月调度模块
├── joint/                  # 联合巡检模块
├── video/                  # 视频监管模块
├── sop/                    # SOP标准执行模块
├── data/                   # 数据中心模块
├── mine/                   # 个人中心与设置模块
└── common/                 # 全局通用页面（错误页、加载页等）
```

## 详细模块说明

### 1. 登录模块 (`src/pages/login/`)
处理用户身份认证及入口逻辑。
- `login.vue`: **登录页** - 应用的主入口，处理账号密码登录。
- `register.vue`: **注册页** - 用户账号注册。
- `forget-password.vue`: **忘记密码** - 找回或重置密码流程。
- `change-password.vue`: **修改密码** - 首次登录强制修改或过期修改密码。
- `guide.vue`: **引导页** - 应用启动时的功能介绍或引导。

### 2. 首页模块 (`src/pages/index/`)
登录后的主要入口。
- `index.vue`: **首页** - 仪表盘及各功能模块入口。
- `ranking/ranking.vue`: **食安指数排名** - 展示食品安全指数的排名情况。

### 3. 日管控模块 (`src/pages/daily/`)
对应“日管控”业务流程。
- **食堂架构选择** (`canteen-structure/`):
  - `area.vue`: **选择片区**。
  - `school-type.vue`: **选择学校类型**。
  - `school.vue`: **选择学校**。
- `record-audit.vue`: **日管控记录审核** - 审核提交的记录。
- `record-rectify.vue`: **整改页** - 处理不合格项的整改。
- `checklist-submit.vue`: **检查表提交** - 填写并提交日管控检查表。
- `submit-success.vue`: **提交成功** - 提交操作后的反馈页。

### 4. 周排查模块 (`src/pages/weekly/`)
对应“周排查”业务流程。
- **食堂架构选择** (`canteen-structure/`):
  - `area.vue`: **选择片区**。
  - `school-type.vue`: **选择学校类型**。
  - `school.vue`: **选择学校**。
- `record-rectify.vue`: **周排查记录整改**。
- `checklist-submit.vue`: **检查表提交**。
- `submit-success.vue`: **提交成功**。
- `rectify-audit-success.vue`: **审核成功** - 整改审核通过后的反馈页。

### 5. 月调度模块 (`src/pages/monthly/`)
对应“月调度”业务流程。
- `detail.vue`: **月调度详情页** - 查看月度调度详细信息。

### 6. 联合巡检模块 (`src/pages/joint/`)
对应“联合巡检”业务流程。
- **食堂架构选择** (`canteen-structure/`):
  - `area.vue`: **选择片区**。
  - `school-type.vue`: **选择学校类型**。
  - `school.vue`: **选择学校**。
- `record-rectify.vue`: **联合巡检记录整改**。
- `checklist-submit.vue`: **检查表提交**。
- `submit-success.vue`: **提交成功**。
- `rectify-audit-success.vue`: **审核成功**。

### 7. 视频模块 (`src/pages/video/`)
包含“视频查看”和“视频监控中心”业务。
- **食堂架构选择** (`canteen-structure/`):
  - `area.vue`: **选择片区**。
  - `school-type.vue`: **选择学校类型**。
  - `school.vue`: **选择学校**。
- `view.vue`: **视频查看页面** - 实时查看监控视频。
- `checklist-submit.vue`: **视频巡检表提交**。
- `record-rectify.vue`: **视频巡检记录整改**。
- `submit-success.vue`: **提交成功**。
- `rectify-audit-success.vue`: **整改审核成功**。

### 8. SOP模块 (`src/pages/sop/`)
对应“SOP标准执行”业务。
- `completed-tasks.vue`: **已完成任务** - 查看历史完成的任务列表。
- `canteen-sop.vue`: **食堂SOP** - 单个食堂的SOP执行页面。

### 9. 数据中心模块 (`src/pages/data/`)
数据统计与展示。
- `data-center.vue`: **数据中心** - 核心数据仪表盘。

### 10. 我的模块 (`src/pages/mine/`)
用户个人中心与设置。
- `mine.vue`: **我的** - 个人中心主页。
- `settings.vue`: **设置** - 应用通用设置。
- `profile.vue`: **个人资料** - 查看和编辑个人信息。
- `notifications.vue`: **通知** - 系统消息通知列表。
- `change-phone.vue`: **更改手机号码**。
- `change-password.vue`: **修改密码**。

### 11. 公共模块 (`src/pages/common/`)
全局通用功能及异常处理页面。
- `network-error.vue`: **网络异常** - 网络连接失败提示页。
- `account-error.vue`: **账户异常** - 账户状态异常提示页。
- `upgrade-reminder.vue`: **升级提醒** - 版本更新提示。
- `loading.vue`: **数据加载** - 全局加载过渡页。
- `empty-data.vue`: **空数据** - 无数据时的占位页。
- `gestures.vue`: **常用手势** - 手势操作说明。
- `notice.vue`: **公共说明** - 用户协议、隐私政策等说明。
