# app-admin 登录模块后端补充接口清单

## 1. 目标与范围

- 适用端：监管端 App（`app_client = reg_app`）
- 模块范围：登录页、注册页、忘记密码页、重置密码页、退出登录
- 统一响应：`{ "code": number, "msg": string, "data": any }`
- 鉴权约定：
  - 登录/注册/短信发送/密码重置相关：可不带 `Authorization`
  - 其余接口：必须带 `Authorization: Bearer <accessToken>`
  - 统一建议带 `X-App-Client: reg_app`

## 2. 已有接口（前端已接）

- `POST /auth/login`
- `POST /auth/refresh`

## 3. 必补接口（按优先级）

### 3.1 短信验证码发送

- **接口**：`POST /auth/sms/send`
- **用途**：登录验证码、注册验证码、忘记密码验证码
- **请求体**

```json
{
  "mobile": "13800000000",
  "scene": "LOGIN",
  "app_client": "reg_app"
}
```

- **响应体**

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "bizNo": "sms_20260318_xxx",
    "expireSeconds": 300,
    "retryAfterSeconds": 60
  }
}
```

***

### 3.2 短信验证码登录

- **接口**：`POST /auth/login/sms`
- **用途**：登录页“手机验证码登录”
- **请求体**

```json
{
  "mobile": "13800000000",
  "code": "123456",
  "bizNo": "sms_20260318_xxx",
  "app_client": "reg_app"
}
```

- **响应体**（同 `/auth/login`）

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "accessToken": "xxx",
    "refreshToken": "xxx",
    "userInfo": {
      "id": 1,
      "roleType": "REGULATOR",
      "tenantId": 1
    }
  }
}
```

***

### 3.3 注册

- **接口**：`POST /auth/register`
- **用途**：注册页“立即注册”
- **请求体**

```json
{
  "mobile": "13800000000",
  "code": "123456",
  "bizNo": "sms_20260318_xxx",
  "password": "plain_or_encrypted",
  "inviteCode": "INVITE_ABC",
  "app_client": "reg_app"
}
```

- **响应体**

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "userId": 1001
  }
}
```

***

### 3.4 忘记密码-验证码校验

- **接口**：`POST /auth/password/verify-code`
- **用途**：忘记密码页“下一步”
- **请求体**

```json
{
  "mobile": "13800000000",
  "code": "123456",
  "bizNo": "sms_20260318_xxx",
  "app_client": "reg_app"
}
```

- **响应体**

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "resetToken": "reset_xxx",
    "expireSeconds": 600
  }
}
```

***

### 3.5 忘记密码-重置密码

- **接口**：`POST /auth/password/reset`
- **用途**：重置密码页“确定”
- **请求体**

```json
{
  "mobile": "13800000000",
  "newPassword": "new_password",
  "resetToken": "reset_xxx",
  "app_client": "reg_app"
}
```

- **响应体**

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "updated": true
  }
}
```

***

### 3.6 获取当前用户信息

- **接口**：`GET /auth/me`
- **用途**：登录后拉取用户信息，补充页面展示与业务分流
- **请求头**：
  - `Authorization: Bearer <accessToken>`
  - `X-App-Client: reg_app`
- **响应体**

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "id": 1,
    "username": "admin",
    "realName": "管理员",
    "mobile": "13800000000",
    "roleType": "REGULATOR",
    "tenantId": 1,
    "status": "ACTIVE"
  }
}
```

***

### 3.7 退出登录

- **接口**：`POST /auth/logout`
- **用途**：设置页退出登录，服务端回收会话
- **请求头**：
  - `Authorization: Bearer <accessToken>`
  - `X-App-Client: reg_app`
- **请求体**

```json
{
  "refreshToken": "xxx"
}
```

- **响应体**

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "ok": true
  }
}
```

## 4. 错误码补充要求

- `400` 参数错误（手机号格式、邀请码错误、密码不合规）
- `401` 未认证/令牌无效
- `403` 无权限/端侧不匹配
- `404` 资源不存在
- `409` 冲突（手机号已注册）
- `429` 验证码发送频率限制
- `46001` 验证码错误
- `46002` 验证码过期
- `46003` 验证码验证次数超限

## 5. 前端页面与接口映射

- 登录页 `login.vue`
  - 账号密码登录：`POST /auth/login`（已接）
  - 验证码登录：`POST /auth/login/sms`（待补）
  - 获取验证码：`POST /auth/sms/send`（待补）
- 注册页 `register.vue`
  - 获取验证码：`POST /auth/sms/send`（scene=REGISTER）
  - 提交注册：`POST /auth/register`
- 忘记密码页 `forget-password.vue`
  - 获取验证码：`POST /auth/sms/send`（scene=RESET\_PASSWORD）
  - 校验验证码：`POST /auth/password/verify-code`
- 重置密码页 `change-password.vue`
  - 提交新密码：`POST /auth/password/reset`
- 全局会话
  - 续期：`POST /auth/refresh`（已接）
  - 登出：`POST /auth/logout`
  - 用户信息：`GET /auth/me`

## 6. 联调前置条件

- 后端返回字段名保持稳定（如 `accessToken`、`refreshToken`、`userInfo`）
- `X-App-Client` 对 `reg_app` 放行
- 测试环境提供可用账号、邀请码、短信通道（或测试白名单验证码）
- 明确密码传输规则（明文 / `b64:` / 加密算法）
